# 平面设计师方案产出指南 · 领域方法论

> 本文档是 `graphic-design-guide` 技能的领域知识库。

---

## 一、场景识别

| 场景 | 名称 | 判断条件（满足任一） | 产出量 | 预估周期 |
|------|------|---------------------|--------|---------|
| 场景一 | 0→1 品牌识别设计 | 新品牌/子品牌、无现有VI规范、需定义品牌视觉语言 | 10-12类 | 3-6周 |
| 场景二 | 营销活动物料 | 主题活动/节日/促销、多媒介(线上+线下)、多尺寸适配 | 6-8类 | 1-2周 |
| 场景三 | 单件物料制作 | 单张海报/单个Banner/单个图标/单张名片 | 2-3类 | 1-2天 |
| 场景四 | 品牌焕新/VI升级 | 品牌重塑(Rebranding)、LOGO/色彩/字体变更、涉及全渠道物料切换 | 8-10类 | 3-8周 |
| 场景五 | 风格探索/Mood Board | 方向探索、灵感收集、风格提案 | 3-4类 | 3-5天 |

### 识别流程

```
是否新品牌/无现有VI？ → 是 → 场景一
    ↓否
是否品牌焕新/VI大版本升级？ → 是 → 场景四
    ↓否
是否为风格探索/灵感收集？ → 是 → 场景五
    ↓否
是否涉及多媒介(印刷+数字)/多尺寸/系列活动？ → 是 → 场景二
    ↓否
→ 场景三
```

## 二、场景一：0→1 品牌识别设计（12类产出）

### 调研阶段
1. **品牌调研报告** — 行业视觉趋势、竞品品牌分析（LOGO/色彩/字体/调性）、目标受众审美偏好
2. **品牌策略文档** — 品牌定位、核心价值、品牌个性关键词、品牌故事框架

### 规范阶段
3. **LOGO设计** — 主标(横版/竖版)、图标标、文字标、组合方式、最小尺寸、安全区、反白/单色版本、应用场景示例
4. **色彩体系** — 主色(含色值CMYK/RGB/HEX)、辅色、中性色、色彩使用比例（60-30-10法则）、色彩禁忌
5. **字体体系** — 中文字体(标题/正文/装饰)、英文字体配对、字号层级(6级)、字重使用规范、行高/字间距
6. **辅助图形/Icon** — 品牌辅助图形(平铺/渐变/描边)、图标系统(线性/面性)、图文结合规范

### 应用阶段
7. **品牌手册** — 以上所有规范的统一呈现、品牌故事页、禁止使用示例
8. **办公VI应用** — 名片(中英文版)、信封/信纸、PPT模板(封面/内页/结尾)、邮件签名
9. **线上VI应用** — 社交媒体头像/封面(微信/抖音/小红书/知乎)、公众号排版规范、网站Favicon、App Icon
10. **线下VI应用** — 手提袋、包装盒、工作牌/工服、户外广告(公交站/灯箱)、展会物料
11. **品牌延展方案** — IP形象/吉祥物、品牌视频片头/片尾、品牌声音(如有)
12. **品牌资产交付包** — 源文件(AI/PSD/Figma)、导出文件(PNG/SVG/PDF)、字体文件(含授权说明)、资产目录清单

## 三、场景二：营销活动物料（6-8类产出）

1. **活动Brief理解** — 活动主题、目标受众、核心利益点、品牌调性约束
2. **主KV设计** — 活动主题视觉、核心视觉元素(图形/人物/产品)、多版本(横版/竖版/方形)
3. **线上物料** — Banner(各平台尺寸)、社交媒体海报(9:16/1:1/16:9)、H5落地页头图、邮件EDM模板、开屏广告、信息流广告素材
4. **线下物料** — 海报(多尺寸)、易拉宝/X展架、DM单页/折页、台卡/桌贴、吊旗/横幅
5. **物料延展清单** — 所有物料的尺寸表、平台规范对照、文案适配
6. **字体与图片版权确认** — 商用字体授权、图片素材授权、避免侵权
7. **色彩模式确认** — 印刷物料CMYK + 300dpi、数字物料RGB + 72dpi、PANTONE色(如需专色)
8. **工艺说明**（如需印刷）— 纸张类型/克重、覆膜(亮膜/哑膜)、烫金/UV/凹凸/模切

---

## 四、场景三：单件物料制作（2-3类产出）

1. **需求规格** — 物料类型、尺寸(像素/mm/inch)、用途(印刷/屏幕/社交媒体)、品牌约束(LOGO/色彩/字体)
2. **设计方案** — 版式构图、配色方案、字体选择、视觉焦点、COT(行动号召)
3. **交付规格** — 格式(PNG/JPG/PDF/AI)、分辨率、色彩模式、出血(印刷品+3mm)、安全区

---

## 五、场景四：品牌焕新/VI升级（8-10类产出）

1. **品牌焕新评估报告** — 焕新动机、现有品牌问题分析、焕新目标
2. **新旧对比方案** — LOGO/色彩/字体/图形 旧→新对照、演变逻辑说明
3. **新版品牌手册** — 更新所有视觉规范（LOGO/色彩/字体/图形/应用示例）
4. **切换计划** — 分阶段切换（Digital First→Print Later→空间环境）、优先级矩阵（高频触点优先）
5. **存量物料替换清单** — 所有需更新物料的清单（按渠道/类型/优先级）、更新进度跟踪
6. **品牌焕新通告** — 内部通告模板（含新旧对比+使用规范）、外部发布（社交媒体/官网/媒体稿）
7. **过渡期双标管理** — 旧版物料使用截止日期、新版物料上线时间表
8. **设计师交接文档** — 源文件清单、字体/素材授权清单、合作供应商信息
9. **新品牌培训材料** — 品牌规范培训PPT、常见问题FAQ、使用自查清单
10. **焕新效果评估** — 关键触点新旧对比截图、用户/客户反馈收集方案

---

## 六、场景五：风格探索/Mood Board（3-4类产出）

1. **探索Brief** — 项目背景、风格方向关键词、参考品牌/设计师
2. **Mood Board（2-3套）** — 每套含配色方案(5-8色)、字体方向(2-3组中文+英文)、质感/图形风格参考图、设计关键词标签
3. **风格提案对比** — 各套风格的适用场景、优势劣势、推荐排序
4. **设计趋势分析** — 当前流行风格(新丑/酸性/弥散光/毛玻璃/Y2K/极简等)、行业趋势(科技/消费/文化等细分)

## 七、通用交付标准

### 色彩规范
| 用途 | 色彩模式 | 色值格式 |
|------|---------|---------|
| 屏幕显示(Web/App/PPT) | RGB | HEX(#RRGGBB) 或 RGB(R,G,B) |
| 印刷(CMYK) | CMYK | C,M,Y,K(百分比) |
| 专色(特种印刷) | PANTONE | PANTONE色号 |

### 分辨率规范
| 用途 | 分辨率 | 说明 |
|------|--------|------|
| 屏幕显示 | 72 PPI | 像素=物理尺寸×72 |
| 普通印刷 | 300 DPI | 海报/传单/名片 |
| 高清印刷 | 350+ DPI | 画册/精美印刷品 |
| 喷绘/户外 | 72-150 DPI | 视距决定精度 |

### 出血与安全区
- 印刷品出血：3mm（常规），5mm（精装画册）
- 安全区：距裁切边 ≥ 5mm（重要内容不要靠近边缘）
- 社交媒体：头像/封面注意各平台的圆角/遮挡区域

### 图片版权检查清单
- [ ] 图片是否为自有/原创
- [ ] 是否使用免版权图库（Unsplash/Pexels/Pixabay）——注意商业用途限制
- [ ] 如使用商业图库（Shutterstock/视觉中国），确认授权类型（标准/RF/扩展）
- [ ] 字体是否获得商用授权（思源免费商用 / 付费字体需确认授权）
- [ ] 人物肖像是否获得肖像权授权

### 物料命名规范
- 格式：`{项目}_{物料类型}_{尺寸}_{版本}.{格式}`
- 示例：`618大促_主KV_横版1920x1080_v2.psd`
- 英文名可选：`618_Sale_MainKV_1920x1080_v2.psd`

### 源文件组织规范（以Figma/Illustrator为例）
- 图层面板：命名清晰、分组合理、无"Layer 1""Rectangle 3"等默认名
- 画板命名：包含尺寸信息
- 文字：全部转曲（交付印刷）/保留可编辑（交付客户）

## 九、产出质量检查清单

### 通用检查
- [ ] 场景判断正确并已确认
- [ ] 产出清单完整
- [ ] 命名与文件组织符合规范

### 品牌一致性
- [ ] 色彩使用正确（主色/辅色比例，无色彩禁忌）
- [ ] 字体使用正确（已获商用授权）
- [ ] LOGO使用正确（尺寸/安全区/背景对比度）
- [ ] 辅助图形使用恰当
- [ ] 品牌调性一致（视觉风格与品牌定位匹配）

### 技术规范
- [ ] 色彩模式正确（印刷CMYK/屏幕RGB）
- [ ] 分辨率达标（印刷300DPI/屏幕72PPI）
- [ ] 出血与安全区正确（印刷品+3mm出血）
- [ ] 字体转曲（印刷交付）/ 可编辑（客户源文件交付）

### 版权合规
- [ ] 图片版权已确认（自有/免版权/已购授权）
- [ ] 字体商用授权已确认
- [ ] 人物肖像权已确认（如涉及）

### 交付
- [ ] 源文件完整（AI/PSD/Figma等）
- [ ] 导出文件齐全（各平台各尺寸）
- [ ] 字体文件随附（或注明来源与授权）
- [ ] 交付清单文档
